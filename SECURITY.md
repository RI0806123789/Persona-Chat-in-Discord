# セキュリティポリシー

## 脆弱性の報告について

当プロジェクトでは「Persona-Chat-in-Discord」のセキュリティを非常に重要視しています。セキュリティ上の脆弱性を発見した場合は、以下の手順に従ってご報告をお願いいたします。

**※ セキュリティ上の脆弱性を公開のGitHub Issueで報告しないでください。**

1. **報告方法:**
   脆弱性の報告は、[liangzhenyiteng5@gmail.com] 宛てにメールを送信してください。

2. **記載していただきたい内容:**
   - 脆弱性の詳細な説明
   - 問題を再現するための具体的な手順（Gemini APIやドキュメント機能に関連するものの場合は、使用したプロンプトや添付ファイルも含めてください）
   - その脆弱性によって引き起こされる可能性のある影響
